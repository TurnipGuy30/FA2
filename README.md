
# FA2 - Prototype Digital Solution

For the imagined client company *Timber Fencing Supplies*.

## Flask application

Runs on Windows 10; requires Python 3

```bat
$ git clone https://github.com/TurnipGuy30/FA2.git
$ cd FA2
$ venv
(.venv) $ run
```
